# Opportunity Types Reference

This document defines the five core opportunity types and their characteristics.

---

## 1. Product Integration

**Description:** Vendor has launched AI productivity tool (content generation, code assistance) that helps teams BUILD faster but doesn't ensure resulting websites WORK for AI agents.

**Characteristics:**
- Vendor has recent AI feature launch
- Focus on internal productivity, not agent-accessible output
- Gap between "build fast" and "works for agents"
- CMS/DXP platforms most common (Kentico, Adobe, Contentful, etc.)

**Typical engagement:**
- 6-9 month product development partnership
- MCP server extensions or native CMS features
- Pattern library integration
- Component templates with agent-ready defaults

**Value range:** £50,000-250,000 initial + revenue share on premium features

**Examples:**
- Kentico AIRA (content generation without Schema.org)
- Adobe Sensei (asset tagging without structured data exposure)
- Contentful AI (content optimization without semantic HTML enforcement)

**Deliverables:**
- MCP server or plugin
- Agent-ready component library (50+ patterns)
- Validation rules and testing suite
- Documentation and training materials

---

## 2. Platform Validation

**Description:** Vendor's customers don't know if their websites work for AI agents. Need measurement, scoring, and automated recommendations integrated into vendor's platform.

**Characteristics:**
- Vendor has mature customer base
- Customers asking "does our site work for agents?"
- No built-in validation or measurement
- Opportunity for platform integration

**Typical engagement:**
- Platform integration partnership (4-6 months)
- Web Audit Suite integration with vendor's admin interface
- Dashboard with agent-readiness metrics
- Usage-based revenue model (per-site audits)

**Value range:** £75,000-150,000 integration + £500-2,000 per site audited

**Examples:**
- Kentico "Agent Readiness Score" dashboard
- Adobe Experience Cloud validation layer
- Contentstack agent-compatibility checker

**Deliverables:**
- Platform plugin/integration
- Admin dashboard with metrics
- Automated reporting and alerts
- API integration documentation

---

## 3. Partner Training

**Description:** Vendor's agency ecosystem needs agent-readiness expertise to serve clients. Training, certification, toolkit, and revenue share model.

**Characteristics:**
- Vendor has established partner programme
- Partners deliver implementations but lack agent-readiness knowledge
- Partners need differentiation vs competitors
- Vendor wants ecosystem strengthening

**Typical engagement:**
- Partner enablement programme (3-4 months setup)
- 2-day training curriculum
- Certification and badging
- Quarterly workshops and updates

**Value range:** £25,000-50,000 programme development + £2,000-5,000 per partner trained + 20-30% revenue share

**Examples:**
- Kentico Partner Accelerator enhancement
- Adobe Solution Partner training
- Contentful agency certification

**Deliverables:**
- Training curriculum (2-day workshop)
- Certification assessment
- Partner toolkit (audit templates, proposals, pricing)
- Revenue share agreement

---

## 4. Enterprise Audit

**Description:** Large organization with complex multi-site estate needs comprehensive agent-readiness assessment across entire web portfolio.

**Characteristics:**
- 50-500+ websites
- Multiple brands or business units
- Lack visibility into agent compatibility
- Need prioritized remediation roadmap

**Typical engagement:**
- One-time comprehensive audit (6-12 weeks)
- Optional: Annual retainer for ongoing monitoring
- Executive reporting and technical specifications
- Implementation roadmap with effort estimates

**Value range:** £25,000-100,000 depending on site count + optional £5,000-15,000/month retainer

**Examples:**
- BBC multi-site estate (200+ sites)
- Nissan-Renault portfolio audit
- Financial services firm (HSBC, Barclays) compliance
- Retail chains (John Lewis, Tesco)

**Deliverables:**
- Automated audit across all sites
- Executive summary with site-by-site scoring
- Detailed technical reports
- 12-24 month remediation roadmap
- Priority-based implementation guidance

---

## 5. Strategic Advisory

**Description:** Vendor needs strategic direction on agent-compatibility across entire product portfolio. Long-term partnership for competitive positioning and thought leadership.

**Characteristics:**
- Vendor at strategic crossroads
- Multiple products needing agent-readiness strategy
- Competitive pressure from other vendors
- Need external validation and credibility

**Typical engagement:**
- 12-month retainer with quarterly reviews
- Product roadmap input
- Competitive analysis
- Executive presentations

**Value range:** £150,000-400,000 annually

**Examples:**
- Kentico agent-readiness roadmap
- Adobe next-generation Experience Cloud strategy
- Contentful composable DXP positioning

**Deliverables:**
- Strategic roadmap (24-month horizon)
- Product requirements specifications
- Competitive positioning analysis
- Thought leadership content (whitepapers, case studies)
- Executive board presentations

## Opportunity Type Decision Tree

**Start here:** What's the primary need?

```
1. Does vendor have AI feature but output isn't agent-ready?
   YES → Product Integration
   NO → Continue

2. Do vendor's customers need measurement/validation?
   YES → Platform Validation
   NO → Continue

3. Does vendor have partner ecosystem needing training?
   YES → Partner Training
   NO → Continue

4. Is this a large org needing multi-site assessment?
   YES → Enterprise Audit
   NO → Continue

5. Does vendor need strategic roadmap and positioning?
   YES → Strategic Advisory
   NO → Opportunity may not fit our model
```

## Combining Opportunity Types

Many opportunities involve multiple types. Present as phased engagement:

**Example: Kentico**
- Phase 1: Strategic Advisory (6 months, £150k) - Define roadmap
- Phase 2: Product Integration (9 months, £200k) - Build AIRA/KentiCopilot extensions
- Phase 3: Platform Validation (6 months, £100k) - Integrate scoring dashboard
- Phase 4: Partner Training (ongoing, £50k setup + revenue share) - Enable agencies

**Total potential value:** £500k+ over 24 months

**Example: Enterprise (BBC)**
- Phase 1: Enterprise Audit (3 months, £75k) - Assess 200+ sites
- Phase 2: Strategic Advisory (12 months, £150k) - Ongoing remediation guidance
- Phase 3: Partner Training (optional) - Train internal dev teams

**Total potential value:** £225k+ over 12-15 months

## Opportunity Qualification Checklist

Before documenting opportunity, verify:

**Gap is clear:**
- [ ] Specific gap identified (not vague "needs help")
- [ ] Evidence gap exists (vendor launch, customer questions, competitive analysis)
- [ ] Gap relates to book's scope (agent-website compatibility)

**Timing is reasonable:**
- [ ] Market urgency exists (not speculative future need)
- [ ] Competitive window identified (3-6 months typical)
- [ ] Vendor receptive (recent launches, market awareness, partnership openness)

**Value is estimable:**
- [ ] Can map to one of five opportunity types
- [ ] Similar examples exist in database for comparison
- [ ] Engagement scope definable
- [ ] ROI calculable for vendor/client

**Approach is feasible:**
- [ ] Decision-makers identifiable
- [ ] Outreach path clear (warm contacts, cold email, thought leadership)
- [ ] Supporting materials can be created
- [ ] Differentiation from competitors articulated

**If all boxes checked:** Document opportunity and generate materials

**If gaps exist:** Research further or decline opportunity

## Revenue Model by Opportunity Type

| Type | Engagement Model | Initial Revenue | Recurring Revenue | Revenue Share |
| ---- | ---------------- | --------------- | ----------------- | ------------- |
| **Product Integration** | Fixed price + revenue share | £50k-250k | Optional support: £5k-10k/month | 10-20% on premium features |
| **Platform Validation** | Integration + usage fees | £75k-150k | £500-2k per site audited | N/A (usage-based) |
| **Partner Training** | Setup + per-partner + share | £25k-50k setup | £2k-5k per partner trained | 20-30% of audit revenue |
| **Enterprise Audit** | Fixed price + optional retainer | £25k-100k | Optional: £5k-15k/month | N/A (fixed pricing) |
| **Strategic Advisory** | Annual retainer | £150k-400k/year | Built into retainer | N/A (retainer model) |

**Margins:**
- Audit work: 50-70% (mostly automated + report writing)
- Product integration: 30-50% (depends on developer rates)
- Training: 40-60% (materials reusable across cohorts)
- Advisory: 60-80% (expertise-based, minimal materials cost)

## Geographic and Market Adjustments

**Adjust pricing based on:**

**Market size:**
- SMB (small business): -30-50%
- Mid-market: Baseline
- Enterprise: +20-50%

**Geography:**
- UK: Baseline
- Western Europe: +10-20%
- US: +20-40%
- APAC: -10% to +20% (varies by country)

**Vendor maturity:**
- Startup: -20-40% (equity or revenue share options)
- Growth stage: Baseline
- Established: +10-30%

**Strategic importance:**
- Reference client potential: -10-20%
- Thought leadership opportunity: -10-20%
- Competitive urgency: +20-40%
