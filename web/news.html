Book Publication and Identity Delegation Project Announcement

The Invisible Users: Designing the Web for AI Agents and Everyone Else is now available. The book examines how modern web design optimised for human users fails for AI agents, and how fixing this benefits everyone.

Alongside the book publication, we're announcing the Universal Identity Delegation Infrastructure project - an open-source initiative to enable portable, user-controlled agent authorisation across platforms.

Key announcements:

  • Complete manuscript published covering 11 chapters on agent-friendly web design
  • Identity delegation proof-of-concept code released as educational reference
  • Collaboration opportunities open for identity systems, OAuth implementations, and agent architectures
  • Development roadmap published for community-driven standards

Visit the book website for full details and access to all materials.

Identity Delegation Project Roadmap Published

The Universal Identity Delegation Infrastructure project roadmap is now available, outlining a five-phase development plan from proof-of-concept through production launch.

The project addresses a critical gap in the AI agent ecosystem: when agents make purchases, businesses lose customer identity and users lose control over authorisations. The infrastructure will provide portable OAuth 2.0 delegation tokens that work across platforms whilst preserving user control.

Phase 1 (Q4 2024): Proof of concept complete - demonstrated in book with Cloudflare Worker example

Phase 2 (Q1 2025): Standards definition in progress - token format, verification protocol, governance model

Phases 3-5 (Q2-Q4 2025): Core infrastructure, integration SDKs, and production launch planned

We're seeking collaborators with expertise in identity systems, OAuth implementations, agent architectures, and open standards development.
